# specs/

Working space for **pre-spec drafting** of web-forth.

Flow:

1. Draft ideas, decisions, and open questions here (this folder).
2. Sharpen fuzzy bits with `/ck:grill` (lands answers into goal/constraints).
3. Promote to the formal spec with `/ck:spec` — which writes **`SPEC.md` at the repo root** (§G goal, §C constraints, §I interfaces, §V invariants, §T tasks, §B bugs).

So: `specs/` = scratch/draft; root `SPEC.md` = the source of truth once `/ck:spec` runs. Keep drafts here readable and link decisions to the eventual §-sections.

- `00-web-forth-overview.md` — vision, locked decisions, architecture, v1 word-set, open questions.
